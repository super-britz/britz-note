#### 文件跳转

#### 权限

* id – 显示用户身份号
* chmod – 更改文件模式
* umask – 设置默认的文件权限
* su – 以另一个用户的身份来运行 shell
* sudo – 以另一个用户的身份来执行命令
* chown – 更改文件所有者
* chgrp – 更改文件组所有权
* passwd – 更改用户密码

```js
-rw-r--r--  1 britz  staff      342  1  8 14:53 theme.txt
//chmod 600 theme.txt 八进制表示法
-rw-------  1 britz  staff      342  1  8 14:53 theme.txt
//chmod u+x theme.txt 符号表示法
-rwxr--r--@  1 britz  staff      342  1  8 14:53 theme.txt
// 第1个字符 d表示目录， -代表文件
// 2-4 当前文件所有者 读、写、执行权限 
// 5-7 用户组
// 8-10 其他人
```
每个八进制数字代表了 3个二进制数字
| Octal八进制 | Binary二进制 | File Mode文件模式 |
| ----------- | ------------ | ----------------- |
| 0           | 000          | ---               |
| 1           | 001          | --x               |
| 2           | 010          | -w-               |
| 3           | 011          | -wx               |
| 4           | 100          | r--               |
| 5           | 101          | r-x               |
| 6           | 110          | rw-               |
| 7           | 111          | rwx               |


| 符号表示法 | 字符位数                                   |
| ---------- | ------------------------------------------ |
| u          | "user"的简写，意思是文件或目录的所有者。   |
| g          | 用户组。                                   |
| o          | "others"的简写，意思是其他所有的人。       |
| a          | "all"的简写，是"u", "g"和“o”三者的联合。 |
`+` 增加权限
`-` 删除权限
`=`重置权限

#### 进程 progress

* ps 报告当前进程快照
	- ps x 展示所有进程
	- ps au 显示较详细的资讯
	- ps aux  显示属于每个用户的进程信息
* top 显示任务
* jobs 列出活跃的任务
* bg 把一个任务放到后台执行
* fg 把一个任务放到前台执行
* kill 给一个进程发送信号

| 名称 | 编号 | 含义                           |
| ---- | ---- | ------------------------------ |
| HUP  | 1    | 终端断线                       |
| INT  | 2    | 中断（同 Ctrl + C）            |
| QUIT | 3    | 退出（同 Ctrl + \）            |
| TERM | 15   | 终止                           |
| KILL | 9    | 强制终止                       |
| CONT | 18   | 继续（与STOP相反， fg/bg命令） |
| STOP | 19   | 暂停（同 Ctrl + Z）            |

* killall 杀死指定名字的进程
* shutdown 关机或重启系统

#### 网络系统

* ping - 发送 ICMP ECHO_REQUEST 软件包到网络主机，一个成功执行的“ping”命令会意味着网络的各个部件（网卡，电缆，路由，网关） 都处于正常的工作状态。
* traceroute - 显示从本地到指定主机 要经过的所有“跳数”的网络流量列表。例如，看一下到达 `www.baidu.com` 网站，需要经过的路由器 `traceroute www.baidu.com`
* netstat - 打印网络连接，路由表，接口统计数据，伪装连接，和多路广播成员
* ftp - 因特网文件传输程序
* wget - 非交互式网络下载器
* ssh - OpenSSH SSH 客户端（远程登录程序 ）
* scp - 被用来复制文件，利用 SSH 加密通道在网络间复制文件，类似cp
* sftp - ftp 程序的安全替代品
